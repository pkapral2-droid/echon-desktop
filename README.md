# Echon Desktop

Desktop app for [Echon](https://echon-voice.com) — privacy-first voice chat, messaging, and screen sharing.

## Platforms

- **Windows** — NSIS installer
- **macOS** — DMG (Intel + Apple Silicon)
- **Linux** — AppImage + .deb

## Download

Get the latest release from [echon-voice.com](https://echon-voice.com) or the [Releases](https://github.com/pkapral2-droid/echon-desktop/releases) page.

## Built With

- [Electron](https://www.electronjs.org/)
- [electron-builder](https://www.electron.build/)
- [uiohook-napi](https://github.com/nickedwards109/uiohook-napi) — global push-to-talk
